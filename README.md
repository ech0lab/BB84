# Implémentation de BB84 avec qiskit

* Arguments : 

```
usage: bb84.py [-h] [-k KEY] [-i] [-c CHECK]

optional arguments:
  -k KEY, --key KEY     Taille de la clé, defaut : 128
  -i, --intercept       Interception du message par Eve
  -c CHECK, --check CHECK
                        Nombre de qubits à vérifier, defaut : 20
```
